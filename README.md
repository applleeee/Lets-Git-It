## 소개

- 개발 동기부여를 위한 깃허브 랭킹 서비스
- let's git it을 통해 자신의 깃허브 점수를 확인하고 친구, 유명인들과 비교해 보세요!
- 개발 관련 커뮤니티(개발 질문, 프로젝트 모집, 채용 정보 등 자유롭게 공유해주세요)
